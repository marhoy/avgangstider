Using the API in other projects
-------------------------------
Avgangstider requires Python 3.7. I recommend using `pyenv <https://github
.com/pyenv/pyenv>`_ for installing and managing different Python versions.

Avgangstider is available on PyPi, so inide the environment where you want
Avgangstider, all you have to do is::

   pip install avgangstider
